# 게시판 기능

## [맛집 리스트]

**필요한 화면** 

1. 맛집리스트 메인 화면(rest_main.html)

- **맛집리스트 메인 화면**
    - 근처(학교기준) 맛집의 위치 정보를 표시하는 맵과 등록된 맛집 리스트 표시
    - 맛집 리스트에 등록되어있는 맛집 항목에는 기본적으로 식당 이름, 주소, 평점, 리뷰 갯수가 표시됨
    - 리스트에 있는 항목 클릭 시 다른 유저가 작성한 해당 맛집의 리뷰, 평점, 좋아요, 싫어요가 표시됨
    - 리스트 항목 구성
        - 해당 맛집 이름
        - 맛집 주소
        - 리뷰 작성자, 리뷰 내용, 평점
        - 다른 유저가 해당 리뷰 좋아요, 싫어요 가능
    - 페이징 기능 ( 1페이지당 글 10개 정렬,  각페이지번호로 페이지이동, 이전,다음 페이지 이동)
    - 검색 기능 (식당 이름, 음식 카테고리, 리뷰 등)
