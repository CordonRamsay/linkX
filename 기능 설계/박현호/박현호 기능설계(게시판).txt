# 게시판 기능

## [자유 게시판]

**필요한 화면** 

1. 게시판 목록 화면(board_list.html)
2. 게시글 상세보기 화면 (board_view.html)
3. 게시글 작성 화면 (board_insert.html)
4. 게시글 수정 화면 (board_update.html)

- **게시판 목록 화면**
    - 해당 게시판의 게시글 리스트(글번호, 제목, 작성자, 작성일, 조회수, 좋아요수) 표시
    - 로그인한 계정으로 작성한 글(본인 글)일 경우 게시글 행 강조 효과 ( 색 변경 )
    - 해당 게시판의 전체 글 개수 카운트 표시
    - 해당 글 클릭 시 **상세보기 화면**으로 이동
    - 글쓰기 버튼 클릭 시 **게시글 작성 화면**으로 이동
    - 댓글 목록 구성
        - 해당 게시글 댓글 수
        - 댓글작성자, 댓글내용, 작성일, 좋아요 기능
    - 페이징 기능 ( 1페이지당 글 10개 정렬,  각페이지번호로 페이지이동, 이전,다음 페이지 이동)
    - 검색 기능 ( 글제목, 작성자 기준으로 검색 가능)
- **게시판 글 상세보기 화면**
    - 상단에 게시글 목록 화면으로 돌아가기 버튼
    - 글제목, 글내용, 작성자닉네임, 작성일, 조회수, 좋아요 수(좋아요 기능포함) 표시
    - 로그인한 계정으로 작성한 글(본인 글)일 경우 수정,삭제 버튼 활성화 / 타인의 글일 경우 비활성화
    - 수정 버튼 클릭 시 **게시글 수정 화면**으로 이동
    - 삭제 버튼 클릭 시 게시글 삭제 후 게시글 목록 화면으로 이동
    - 댓글 목록 표시 & 댓글 작성
        - 해당 게시글의 댓글 총 개수
        - 해당 게시글의 댓글 리스트( 작성자닉네임, 댓글내용, 작성일, 좋아요 수(좋아요 기능 포함)) 표시
        - 댓글 입력 창 ( 본인 닉네임, 입력필드, 댓글추가 버튼)
        - 로그인한 계정으로 작성한 글(본인 글)일 경우 댓글 행 강조 - **게시글 작성 화면**
    - 게시판 종류 선택
    - 글제목, 내용 입력 후 취소/등록 버튼으로
        
        게시글목록화면으로 돌아가거나, 게시글 등록 후 게시글목록화면으로 이동
        

- **게시글 수정 화면**
    - 해당 게시글의 게시판종류, 글제목, 글내용 모두 가져오기
    - 제목이나 내용 수정 후 취소/수정 버튼으로
        
        게시글목록화면으로 돌아가거나, 게시글 수정 후 게시글목록화면으로 이동
