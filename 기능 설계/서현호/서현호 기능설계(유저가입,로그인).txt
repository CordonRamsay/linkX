유저 가입

**필요한 화면**

1.로그인 화면
2.회원가입 화면

#로그인 화면
-ID를 입력하는 텍스트 필드,PASSWORD를 입력하는 텍스트 필드 배치
-텍스트 필드 아래에 로그인 버튼 배치
-버튼 아래 회원가입(하이퍼링크)텍스트
-회원가입 텍스트 클릭 시 회원가입화면으로 이동

#회원가입 화면
-ID를 입력하는 텍스트 필드 배치, 텍스트 필드 옆에 중복 검사 버튼 배치
-중복검사를 클릭시 텍스트필드의 입력한 ID를 기존 데이터 베이스에서 일치하는지 확인 후 일치시 
사용불가 팝업, 일치가 없을시 사용가능 팝업 띄우기
-PASSWORD입력하는 텍스트 필드 배치
-PASSWORD 확인하는 텍스트 필드 배치, "PASSWORD"와 "PASSWORD확인" 텍스트 비교 일치시 가능 표시, 불일치시 재입력 안내 표시
-학생임을 인증하는 인증 서류 첨부 파일 등록 공간 마련
-최하단의 가입 신청 버튼, 클릭 시 "가입신청 되었습니다" 안내 후 로그인 화면으로 복귀
